Portfolio Analysis Tool

The Portfolio Analysis Tool is intended to help store information about different portfolios in one place. The tool accesses stock data to evaluate the current value of the portfolios.

To support the evaluation of the Portfolio Analysis Tool, this website uses cookies to store your information. Cookies are stored on your computer and are accessed by this website to remember your settings. This data is not shared and there is no intent of sharing this data. This data is secured on your computer and not accessable by other computers.

Alpha Vantage is the provider of the stock data. To meet the requirements of the Alpha Vantage terms and conditions, it is best for each user to have their own Alpha Vantage access key.

Please visit this site to claim your key and input it in the Alpha Vantage API Key Text box below.
